# 김家네 Project
쇼핑몰 구매 고객 리워드 시스템

# 프로젝트 소개
- 유저 정보 인증 후 하루 한번 출석 체크
- 일정 출석 횟수를 달성했을 때 해당하는 상품을 수령
> ex) 5 일 출석 -> 현금성 포인트 5000원 | 15 일 출석 -> 에어팟
- 이벤트 관리자가 출석 및 리워드 현황 체크 가능
